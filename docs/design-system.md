# Design System

## UI Direction
Audience: marketers who need clear analytics and confidence cues.

## Visual Principles
- Data first, minimal decorative noise.
- Bold but professional tone.
- Accessible contrast and predictable interactions.

## Tokens
- Typeface: Manrope (UI), Space Grotesk (display)
- Primary accent: teal
- Secondary accent: coral
- Base: neutral warm grays

## Component Baseline
- Use shadcn/ui primitives for layout and forms.
- Extend with custom KPI cards, model result table, sentiment chart shell.
- Use TanStack Query for async state and retry behavior.

## Motion
- Subtle chart reveal animation on report load.
- Staggered card entry for dashboard summary only.
