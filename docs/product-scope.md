# Product Scope

## Phase 1: GEO MVP
In scope:
- User auth and single-workspace onboarding
- Brand profile setup (brand name + industry)
- Intent-based prompt runs against OpenAI, Gemini, Anthropic
- Mention detection and sentiment classification
- Weekly mention-rate and sentiment dashboard

Out of scope:
- Live web crawling and SERP ingestion
- Multi-seat RBAC
- White-label report theming
- Fine-tuned custom models

## Phase 2: Video MVP
In scope after GEO baseline:
- Video link ingestion
- Transcript processing
- Top 3 clip segment recommendation
- Suggested captions and timestamps

Out of scope in phase 2 initial cut:
- Full automated rendering pipeline for social exports
- Team review workflow
